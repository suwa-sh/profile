IT部門のジェネラリスト

IT部門のあらゆる業務を社員の立場で支援しています。

サイトを別プロダクトに載せ替え中です。コンテンツが読みにくい部分がありますが、ご容赦いただけると幸いです。

主な活動分野

現場で関わることが多い活動


  • DX

    DX推進

    「顧客視点の獲得」「市場変化への迅速な対応能力の獲得」「新規デジタルビジネスの創出」に向けた活動を、検討段階からフルサイクルで支援しています。

  • 仮説検証
    x
    アジャイル

    仮説検証型アジャイル開発

    仮説検証とアジャイル開発を組み合わせたものづくりを多く採用しています。仮説立案から、BMLループすべての段階を支援しています。


  • プロセス

    プロセス改善

    AsIs業務プロセスの可視化、問題の検出、ToBe像の立案から、必要になるものづくり、ToBeプロセス適用の伴走まで、すべてのステージを支援しています。


  • 人材育成

    イネーブルメント

    求める成果に必要な行動、行動に必要な知識/スキルの整理段階から、コンテンツの整備、トレーニング実施、実績データ収集から分析まで、すべてのサイクルを支援しています。


  • CS

    カスタマーサクセス

    サクセスロードマップ、ヘルススコアの策定から、データ収集・CTAの仕組みづくり、顧客との伴走まで幅広く支援しています。
